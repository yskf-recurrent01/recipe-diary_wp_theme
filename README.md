# Recipe Diary (WordPress)

## 管理画面での管理対象（クライアント要望）

|管理対象|内容|
|:---|:---|
|**お知らせ**|サイトメンテナンス等のお知らせや定期的にイベントを開催しているので、イベント告知をしたい|
|**レシピ**|旬の食材を使った新しいレシピを自分たちでどんどん追加したい。完成写真も大きく載せたいし、季節に合わせてメイン画像を後から入れ替えることもある。|
|**レシピ詳細情報**| 材料や手順をスマホでキッチンで見ながら料理をしやすいように整理して載せたい。各項目を入力する場所がわかりやすい形式にしてほしい。|
|**レシピカテゴリー**|和食、和食、エスニックなど、その日の気分でレシピを探せるようにジャンル分けをして、探しやすくしたい。|
|**固定ページ（About）**|このサイトを立ち上げた想いや、大切にしているこだわりを伝えたい。今後、スタッフの紹介や写真なども自分たちで適宜更新できるようにしたい。|

### 可能であれば実装してほしい機能
|対象|内容|
|:---|:---|
|**トップページ**|メインビジュアルをフェードで切り替わるスライドショー(カルーセル)にしてほしい。スライドショーに使う画像は3枚固定で、スタッフが変更できるようにしたい。|
